# Secure Blog Management System

## 📌 Project Overview

The **Secure Blog Management System** is a web application developed using **PHP** and **MySQL** as the Final Project for the **ApexPlanet Web Development Internship (Task 5)**. The application enables users to securely manage blog posts through authentication, CRUD operations, search, pagination, and role-based access control.

---

## 🚀 Features

* 🔐 User Registration and Login Authentication
* 🔒 Password Hashing for Secure Authentication
* 👤 Session Management
* 📝 Create, Read, Update, and Delete (CRUD) Blog Posts
* 🔍 Search Posts by Title or Content
* 📄 Pagination for Better Navigation
* ✅ Client-side and Server-side Form Validation
* 🛡️ SQL Injection Prevention using Prepared Statements
* 👥 Role-Based Access Control (Admin & Editor/User)
* 📱 Responsive User Interface
* 💾 MySQL Database Integration

## ⚙️ Installation

1. Install **XAMPP**.
2. Start **Apache** and **MySQL**.
3. Copy the project folder into the **htdocs** directory.
4. Open **phpMyAdmin**.
5. Create a new database.
6. Import the provided SQL file.
7. Update the database connection settings if required.
8. Open your browser and visit:

```
http://localhost/your-project-folder
```
